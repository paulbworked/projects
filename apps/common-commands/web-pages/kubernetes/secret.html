Secret Commands

Stores and Manages sensitive information such as passwords and tokens

Get a list of all secrets behind all namespaces:

kubectl get secrets --all-namespaces
kubectl get secrets -n datadog

Create secret in namespace:

This example is creating a secret holding the API key for Datadog to go with install of Datadog agent via Helm

kubectl create secret generic datadog-secret --from-literal api-key=xxx -n datadog
