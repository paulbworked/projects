Ingress Commands

Manages external access to services in a cluster, typically HTTP/HTTPS

List all Ingress resources across all namespaces:

kubectl get ing --all-namespaces

Get details of a specific ingress route:

kubectl describe ingress -n
Kubectl describe ingress autotest-opra | pods behind

get YAML file of the Ingress:

kubectl get ingress dev2-opra-ingress -n dev2-opra -o yaml
