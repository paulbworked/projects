Namespace Commands

Divides cluster resources between multiple users

Create a namespace:

kubectl create namespace datadog

Get a list of all roles behind all namespaces:

kubectl get roles --all-namespaces

Create secret in namespace:

This example is creating a secret holding the API key for Datadog to go with install of Datadog agent via Helm

kubectl create secret generic datadog-secret --from-literal api-key=xxx -n datadog

Delete namespace:

kubectl delete ns cilium-test
