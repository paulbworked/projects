Useful OPRA Commands

Set context to right cluster:

kubectl config use-context aks-opra-dev-ne

To look at pods running, the label shows what version of pod is running:

kubectl get pods -n opra-dev1 -L opra.tritrials.com/version

What resources are being run under a pod/node:

kubectl describe node aks-apopraqual-28135127-vmss000000
