Azure Commands

Various Basics:

az login
Az account show - show current subscription
Az account subscription list - shows all subscriptions in tenant
az account set --subscription "subscription ID or name"
# get access token for the active subscription
az account get-access-token
# get access token for a specific subscription
az account get-access-token --subscription "subscription ID or name"
az config get defaults
az config set defaults.group=rg-temp-prod
az config set defaults.group= - leave string empty
recover failing firewall policy: Get-AzFirewallPolicy -Name "your_resource_name" -ResourceGroupName "your_resource_group_name" | Set-AzFirewallPolicy
